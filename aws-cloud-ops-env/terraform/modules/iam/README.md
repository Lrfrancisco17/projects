# IAM module  

Overview  

Provides IAM roles and instance profiles for the bastion host and application instances. Grants SSM and CloudWatch permissions.  

What it includes  
 * Bastion IAM role + instance profile  

 * App IAM role + instance profile  

 * Policy attachments for SSM and CloudWatch  

Inputs
 
 * name  

Outputs  
 * bastion_instance_profile  

 * app_instance_profile
